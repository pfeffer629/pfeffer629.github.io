Thinking Styles

10/5/14

Class methods and modules are similar, but they have their differences. They both contain methods and constants, yet the major difference is that modules cannot contain instances. What this means is that everything in the module is fixed, while variables in a class can change.

The time to use modules is when the variables used are constant. If they change, then classes must be used instead.
